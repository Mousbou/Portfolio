Semantic Web представляет собой набор технологий / методов, которые, в долгосрочной перспективе, приведет к реализации «интеллектуальной сети». Первая диаграмма описывает различные слои языков введены в действие с иерархическом порядке каждого из верхних слоев эксплуатировать возможности нижнего слоя. Второй показывают различные технологии, используемые и поставить хронологическую линию.
